## Empfohlene Größe
*   **Format:** Querformat (Landscape)
*   **Auflösung:** ca. 800x600 Pixel oder 600x400 Pixel
*   **Dateigröße:** Möglichst klein halten (z.B. < 300 KB)

Das Spiel lädt diese Bilder automatisch.
